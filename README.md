# Casant Website

A modern Next.js website for Casant with TypeScript, admin dashboard, and comprehensive content management.

## Features

- Admin Dashboard for managing:
  - Contacts
  - Gallery
  - Hero sections
  - HR/Team management
  - Inventory
  - Services
  - Testimonials
  - Weddings
  - Instagram integration
  - Settings

- Public pages:
  - About
  - Contact
  - Portfolio
  - Corporate
  - Careers
  - Services
  - Inventory
  - Weddings

- Features:
  - Responsive design
  - Supabase integration
  - Real-time data management
  - Chat functionality
  - Image uploads
  - Video management

## Getting Started

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
npm install
# or
yarn install
```

### Development

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build

```bash
npm run build
npm start
```
